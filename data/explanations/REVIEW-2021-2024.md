# 2021-2024 Explanation Review

## Scope

- 2021 Spring: 25 questions
- 2022 Spring: 25 questions
- 2023 Spring: 25 questions
- 2024 Spring: 25 questions
- Total: 100 questions and 400 choice explanations

## Review checks

- The correct choice matches the IPA answer PDF.
- Every question has explanations for ア, イ, ウ, and エ.
- The correct-choice rationale answers the question directly.
- Each incorrect-choice rationale identifies the different concept, formula, process, or condition.
- Calculation questions were recalculated from the values in the question.
- Diagram and table questions were checked against the extracted page image and answer key.
- Every explanation is marked `status: reviewed` and `reviewed: true`.

## Corrections made during review

- Re-ran OCR for 2021-2024 with the current question-marker filter. This removed the sample-question contamination previously seen in 2024 Q10.
- Restored the missing text of 2022 Q4 choice イ from the original PDF page.
- Restored the missing text of 2022 Q16 choices ウ and エ from the original PDF page.
- Restored the 2021 Q22 MTBF/MTTR choice table from the original PDF page.
- Added persistent manual patches so these corrections survive future PDF regeneration.
- Added a separate explanation source file for each year and merge support in `prepare_exam.py`.

## Remaining source caveat

The question text is based on image OCR. Minor character errors may remain, such as `JIS 0` instead of `JIS Q`, but the reviewed explanations use the intended technical term. Diagram and table questions should be read together with the displayed image where available.
