# 2015-2019 Explanation Review

## Scope

- 2015 Autumn: 25 questions
- 2016 Autumn: 25 questions
- 2017 Autumn: 25 questions
- 2018 Autumn: 25 questions
- 2019 Autumn: 25 questions
- Total: 125 questions and 500 choice explanations

## Review checks

- The correct choice matches the IPA answer PDF.
- Every question has explanations for ア, イ, ウ, and エ.
- The correct-choice rationale answers the question directly.
- Each incorrect-choice rationale identifies the different concept, formula, process, or condition.
- Calculation questions were recalculated from the values in the question.
- Diagram and table questions were checked against the extracted page image and answer key.
- Repeated questions were compared with the reviewed 2021-2024 explanations.
- Every explanation is marked `status: reviewed` and `reviewed: true`.

## Corrections made during review

- Re-ran OCR for all five exams using the current question-marker filter. This removed the sample-question contamination that had replaced Q10 in every year.
- Restored the complete 2018 Q7 choice ア from the original PDF page.
- Restored the complete 2018 Q10 question text from the original PDF page.
- Restored the missing beginning of 2019 Q3 choice イ.
- Added persistent manual patches so these corrections survive future PDF regeneration.
- Added a separate reviewed explanation source file for every exam.

## Remaining source caveat

The question text is based on image OCR. Minor character errors may remain in the displayed question text. The explanations use the intended technical terms, and figure or table questions should be read together with the displayed image where available.
